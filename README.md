# Valley Tractor Farm Supplies Website

A simple, clean multi-page website for Valley Tractor Farm Supplies featuring image sections, detailed content pages, and a contact form.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Navigation**: Easy-to-use links between Home, Gallery, Details, and Contact pages
- **Landing Image Sections**: Editable home page text with clickable image sections
- **Gallery Page**: Additional image tiles with hover carousel previews
- **Details Page**: Deeper descriptions per section with Contact Us buttons
- **Contact Form**: Dedicated contact page that sends submissions to a configurable email endpoint
- **Professional Styling**: Clean, modern design with a farm-themed color scheme

## File Structure

```
├── index.html      # Landing page with editable text and image sections
├── gallery.html    # Additional image gallery page
├── details.html    # Detailed content sections page
├── contact.html    # Contact page with form submission endpoint
├── styles.css      # Shared styling and hover carousel behavior
└── README.md       # This file
```

## Getting Started

1. Clone the repository
2. Open `index.html` in your web browser
3. Customize content, colors, and branding as needed

## Customization

- **Colors**: Update the color scheme in `styles.css` (currently using green #2c5f2d)
- **Content**: Replace placeholder text in `index.html`, `details.html`, and `contact.html`
- **Images**: Swap sample image URLs with your own image files or hosted images
- **Contact Form**: Set your real domain email in `contact.html` form action (for example `name@yourdomain.com`)
