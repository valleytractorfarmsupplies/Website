# Valley Tractor Farm Supplies Website

A simple, clean website for Valley Tractor Farm Supplies featuring products, information, and contact options.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Navigation**: Easy-to-use navigation menu with smooth scrolling
- **Product Showcase**: Featured products section highlighting main offerings
- **About Section**: Company information and key features
- **Contact Form**: Simple contact form for customer inquiries
- **Professional Styling**: Clean, modern design with a farm-themed color scheme

## File Structure

```
├── index.html      # Main HTML page
├── styles.css      # Styling and layout
└── README.md       # This file
```

## Getting Started

1. Clone the repository
2. Open `index.html` in your web browser
3. Customize content, colors, and branding as needed

## Customization

- **Colors**: Update the color scheme in `styles.css` (currently using green #2c5f2d)
- **Content**: Edit text, product names, and descriptions in `index.html`
- **Images**: Replace emoji icons with actual product images
- **Contact Form**: Connect the contact form to a backend service for email functionality
